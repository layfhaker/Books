# 🚀 German Reading Platform - Стартовый гайд

## Что это?

Интерактивный сайт для чтения немецких текстов с мгновенным переводом по клику.

**Ключевые возможности:**
- 📖 Читаешь текст на немецком
- 🖱️ Кликаешь на непонятное слово/фразу
- 🔤 Видишь русский перевод и объяснения
- 📚 Поддержка нескольких книг и глав
- 📱 Работает на любом устройстве

---

## Быстрый старт (2 минуты)

### 1️⃣ Откройте сайт

Просто откройте файл **`index.html`** в браузере.

```
Правой кнопкой мыши на index.html → Открыть с → Браузер
```

или

```
Перетащите index.html в окно браузера
```

### 2️⃣ Выберите книгу и главу

1. В выпадающем списке "Книга" выберите "1Q84"
2. В выпадающем списке "Глава" выберите "Октябрь"
3. Текст загрузился!

### 3️⃣ Кликните на немецкий текст

Нажмите на любое предложение → откроется окно с переводом

## Как добавить собственный текст?

### Вариант 1: Добавить главу к 1Q84

1. **Откройте** `texts/1q84-part3/chapter-1.json` как пример
2. **Скопируйте** содержимое
3. **Создайте новый файл** `chapter-4.json` в папке `texts/1q84-part3/`
4. **Вставьте** структуру и отредактируйте:
   - Измените `chapterId` на `"chapter-4"`
   - Измените `chapterNumber` на `4`
   - Поменяйте `chapterTitle` на "Ваше название"
   - Замените `content` на ваши предложения
5. **Обновите** `texts/books.json` - добавьте новую главу в массив `chapters`
6. **Обновите** страницу в браузере (Ctrl+R)

### Вариант 2: Добавить новую книгу

1. **Создайте папку** `texts/my-book-id/` (вместо `my-book-id` - ваш ID)
2. **Создайте файл** `chapter-1.json` в этой папке
3. **Скопируйте** структуру из `chapter-1.json` (1q84)
4. **Отредактируйте**:
   - `bookId` - ваш ID (должен совпадать с названием папки)
   - `bookTitle` - название книги
   - `chapterId` - ID главы (chapter-1, chapter-2 и т.д.)
   - `content` - ваши предложения
5. **Обновите** `texts/books.json` - добавьте новую книгу в массив
6. **Обновите** страницу в браузере

## Структура предложения в JSON

```json
{
    "id": 1,
    "de": "Ich bin ein Student.",
    "ru": "Я студент.",
    "type": "sentence",
    "difficulty": "A1",
    "tags": ["verb", "present"],
    "notes": "Глагол 'sein' в настоящем времени"
}
```

**Обязательные поля:**
- `id` - номер (1, 2, 3...)
- `de` - текст на немецком
- `ru` - перевод на русский
- `type` - тип (обычно "sentence")

**Опциональные поля:**
- `difficulty` - уровень (A1, A2, B1, B2, C1, C2)
- `tags` - теги (["verb", "present"] и т.д.)
- `notes` - объяснение грамматики

## Типичные ошибки (как избежать)

❌ **Неправильно:**
```json
"content": { "id": 1, ... }  // Это объект, а не массив!
```

✅ **Правильно:**
```json
"content": [ { "id": 1, ... } ]  // Это массив!
```

---

❌ **Неправильно:**
```json
"createdDate": "26.02.2025"  // Неправильный формат!
```

✅ **Правильно:**
```json
"createdDate": "2025-02-26"  // YYYY-MM-DD
```

---

❌ **Неправильно:**
```json
"content": [
    { "id": 1, ... },
    { "id": 1, ... }  // Дублирующийся ID!
]
```

✅ **Правильно:**
```json
"content": [
    { "id": 1, ... },
    { "id": 2, ... }  // Уникальные ID!
]
```

## Проверка перед использованием

### ✅ Чек-лист:

- [ ] JSON файл валиден (проверьте в https://jsonlint.com/)
- [ ] Все ID в `content` уникальны
- [ ] Дата в формате `YYYY-MM-DD`
- [ ] `chapterId` совпадает в `books.json` и имя файла
- [ ] `bookId` совпадает во всех местах
- [ ] Файл сохранён в формате **UTF-8**
- [ ] Уровень из списка: A1, A2, B1, B2, C1, C2

## Если что-то не работает

### Сайт не загружает главу?

1. **Проверьте имя файла:**
   - Если в `books.json` указано `"chapterId": "chapter-1"`
   - Файл должен быть назван **`chapter-1.json`** (не chapter_1, не Chapter-1)

2. **Проверьте JSON синтаксис:**
   - Откройте файл в https://jsonlint.com/
   - Исправьте ошибки

3. **Проверьте пути:**
   - Структура должна быть: `texts/book-id/chapter-n.json`
   - Проверьте правильность ID

### Выпадающий список пуст?

1. Проверьте, что `texts/books.json` существует
2. Проверьте, что массив `chapters` не пустой
3. Перезагрузите страницу (Ctrl+R)

### Модальное окно не открывается?

1. Обновите страницу (Ctrl+Shift+R - полная перезагрузка)
2. Откройте консоль (F12 → Console) и проверьте ошибки
