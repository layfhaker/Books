# German Reading Platform

Интерактивный сайт для чтения текстов на немецком языке с интерактивным переводом.

## Структура проекта

```
/
├── index.html          # Основной HTML файл
├── style.css           # Стили
├── script.js           # JavaScript логика
├── texts/
│   ├── books.json      # Список всех книг и глав
│   └── 1q84-part3/
│       ├── chapter-1.json
│       ├── chapter-2.json
│       └── chapter-3.json
└── README.md           # Этот файл
```

## Использование

### 1. Запуск сайта

Просто откройте `index.html` в браузере. Сайт полностью статичный и работает без сервера.

### 2. Добавление новых книг

#### Шаг 1: Обновите `texts/books.json`

```json
[
    {
        "bookId": "my-book-id",
        "bookTitle": "Название Книги",
        "bookAuthor": "Автор",
        "description": "Краткое описание",
        "chapters": [
            {
                "chapterId": "chapter-1",
                "chapterNumber": 1,
                "chapterTitle": "Название главы"
            },
            {
                "chapterId": "chapter-2",
                "chapterNumber": 2,
                "chapterTitle": "Вторая глава"
            }
        ]
    }
]
```

#### Шаг 2: Создайте папку для книги

```
texts/my-book-id/
```

#### Шаг 3: Добавьте JSON файлы глав

Для каждой главы создайте файл `chapter-N.json`.

### 3. Структура JSON файла главы

```json
{
    "metadata": {
        "bookId": "1q84-part3",           // Должен совпадать с ID в books.json
        "bookTitle": "1Q84",              // Название книги
        "bookAuthor": "Харуки Мураками",  // Автор
        "chapterId": "chapter-1",         // Должен совпадать с ID в books.json
        "chapterNumber": 1,               // Номер главы (число)
        "chapterTitle": "Октябрь",        // Название главы (опционально)
        "level": "A2",                    // Уровень (A1, A2, B1, B2, C1, C2)
        "language": "de",                 // Язык (всегда "de")
        "createdDate": "2025-02-26",      // Дата в формате YYYY-MM-DD
        "description": "Описание"         // Опционально
    },
    "settings": {
        "showWordCount": true,            // Показывать количество слов
        "showDifficulty": true,           // Показывать сложность слов
        "allowNotes": true                // Показывать примечания
    },
    "content": [
        {
            "id": 1,                           // Уникальный ID (число)
            "de": "Текст на немецком языке.",  // Текст на немецком
            "ru": "Текст на русском языке.",   // Перевод на русский
            "type": "sentence",                // Тип: sentence, paragraph, phrase
            "difficulty": "A2",                // Сложность (опционально)
            "tags": ["verb", "present"],       // Теги для категоризации (опционально)
            "notes": "Примечание"              // Дополнительные замечания (опционально, может быть null)
        }
    ]
}
```

## Правила для JSON

### Обязательные поля в metadata:
- `bookId` - ID книги (slug-формат, например: "1q84-part3")
- `bookTitle` - Название книги
- `chapterId` - ID главы (формат: "chapter-N")
- `chapterNumber` - Номер главы (число)
- `level` - Уровень сложности (A1, A2, B1, B2, C1, C2)
- `language` - Язык перевода (всегда "de")
- `createdDate` - Дата (YYYY-MM-DD)

### Обязательные поля в content:
- `id` - Уникальный номер (число, начиная с 1, по порядку)
- `de` - Текст на немецком (не более 500 символов)
- `ru` - Перевод на русский (не более 500 символов)
- `type` - Тип содержимого (sentence, paragraph, phrase)

### Опциональные поля в content:
- `difficulty` - Сложность элемента (A1, A2, B1, B2, C1, C2)
- `tags` - Массив тегов для классификации (например: ["verb", "present", "Präsens"])
- `notes` - Дополнительные объяснения грамматики или контекста

## Функциональность

### 1. Выбор книги и главы
- Выпадающий список для выбора книги
- Автоматическое отображение доступных глав
- Кнопки "Следующая" и "Предыдущая" для быстрой навигации

### 2. Интерактивный перевод
- Кликните на немецкий текст → откроется модальное окно с переводом
- Отображение примечаний (если есть)
- Отображение тегов для контекста

### 3. Метаданные
- Отображение уровня сложности (A2, B1 и т.д.)
- Подсчёт слов в главе
- Описание главы

### 4. Адаптивный дизайн
- Полная поддержка мобильных устройств
- Удобный интерфейс на всех размерах экранов

## Валидация JSON

Перед добавлением файла убедитесь, что JSON валиден:

1. Используйте https://jsonlint.com/
2. Проверьте, что все ID уникальны
3. Убедитесь, что все обязательные поля присутствуют
4. Проверьте формат дат (YYYY-MM-DD)

## Советы

### 1. Организация глав
- Используйте понятные chapterId (chapter-1, chapter-2 и т.д.)
- Номера должны совпадать с порядком в books.json

### 2. Перевод качества
- Перевод должен быть дословным (не литературным)
- Это помощь в обучении, а не артистическая адаптация

### 3. Примечания и теги
- Используйте примечания для объяснения грамматики
- Теги помогают структурировать материал

### 4. Уровни сложности
- A1 - Начинающий
- A2 - Элементарный (твой уровень)
- B1 - Средний
- B2 - Выше среднего
- C1 - Продвинутый
- C2 - Высокий

## Возможные проблемы

### Глава не загружается
- Проверьте, что файл находится в правильной папке
- Убедитесь, что `chapterId` совпадает в books.json и имя файла
- Проверьте JSON на ошибки синтаксиса

### Модальное окно не открывается
- Убедитесь, что класс `german-text` есть на элементе
- Проверьте браузерную консоль на ошибки

### Выпадающий список глав пуст
- Проверьте, что массив `chapters` не пустой в books.json
- Убедитесь, что структура books.json корректна
